# Proyecto de Grafos

## Descripción
Implementación de estructuras de datos de grafos con algoritmos clásicos para procesamiento y análisis de redes.

## Tecnologías Utilizadas
- **JavaScript**: Implementación de algoritmos
- **HTML/CSS**: Visualización interactiva
- **Canvas API**: Renderizado de grafos
- **Algoritmos**: BFS, DFS, Dijkstra, Floyd-Warshall

## Requisitos Previos
- Navegador web moderno (Chrome, Firefox, Safari)
- No requiere instalación de dependencias externas

## Instalación

1. Clona el repositorio:
```bash
git clone https://github.com/jorgerankov/Proyectos.git
cd Proyectos/Grafos
```

2. Abre directamente en el navegador:
```bash
python -m http.server 8000
```

O simplemente abre `index.html` en tu navegador.

## Uso
1. Crea nodos haciendo clic en el canvas
2. Conecta nodos para formar aristas
3. Selecciona un algoritmo (BFS, DFS, Dijkstra)
4. Visualiza el resultado de la ejecución

## Estructura del Proyecto
- `/index.html` - Interfaz principal
- `/js/graph.js` - Clase Graph
- `/js/algorithms.js` - Implementación de algoritmos
- `/css/styles.css` - Estilos

## Algoritmos Implementados
- BFS (Breadth-First Search)
- DFS (Depth-First Search)
- Dijkstra
- Floyd-Warshall
