# ChatBot con AI

## Descripción
Proyecto de chatbot inteligente implementado con tecnologías de inteligencia artificial y procesamiento de lenguaje natural.

## Tecnologías Utilizadas
- **JavaScript**: Lógica principal y manejo de eventos
- **HTML/CSS**: Interfaz de usuario
- **APIs de IA**: Integración con servicios de procesamiento de lenguaje natural
- **Node.js**: Backend

## Requisitos Previos
- Node.js (versión 14 o superior)
- npm o yarn
- Conexión a internet para APIs de IA

## Instalación

1. Clona el repositorio:
```bash
git clone https://github.com/jorgerankov/Proyectos.git
cd Proyectos/ChatBot-AI
```

2. Instala las dependencias:
```bash
npm install
```

3. Configura las variables de entorno:
```bash
cp .env.example .env
```

4. Inicia la aplicación:
```bash
npm start
```

## Uso
Accede a la interfaz y escribe mensajes para interactuar con el chatbot. El sistema procesará tu entrada y generará respuestas contextuales.

## Estructura del Proyecto
- `/src` - Código fuente
- `/public` - Archivos estáticos (HTML, CSS)
- `/config` - Configuraciones
